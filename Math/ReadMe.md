- [自分割数](/Math/self_divide_number.cpp)

 > 自分割数是一个可被其包含的每个数字整除的数字。   
   例如，128是自分割数，因为128％1 == 0,128％2 == 0，128％8 == 0。   
   此外，不允许自分割数包含数字零。   
   给定下限和上限数字，输出包含每个可能的自分割数列表，如果可能，包括边界。
