#### 题目描述
给定一个 `n*n` 的矩阵，求其最大子矩阵和  

> 	input:  
	4  
	0 -2 -7 0   
	9 2 -6 2   
	-4 1 -4 1   
	-1 8 0 -2   
	output:  
	15  
	explanation:  
	9 2   
	-4 1   
	-1 8   
	the above sub-matrix has a max sum of 15. 

#### 解题思路
将求一个矩阵的最大子矩阵和转化为求一个数组的最大子数组和  
求最大子数组和，参考：[具有最大和的子数组](/Array/max_subarray.md)  
矩阵的每行可以看成一个一维数组  
先单独求每行的最大子数组和，此时子矩阵的行数为1  
然后，把第i行后的各行对应列的元素加到第i行对应的列，每加一行，就求一次当前叠加行的最大子数组和  

#### 代码实现

```
#include<iostream>

using namespace std;

int m[101][101];

int main()
{
	int n, max, i, j, k, tmp;

	while(cin >> n)
	{
		max = -10000;

		// 输入每行元素时，求出每行的最大子数组和
		for(i=0; i<n; ++i)
		{
			tmp = 0;
			for(j=0; j<n; ++j)
			{
				cin >> m[i][j];
				if(tmp > 0)	
					tmp += m[i][j];
				else 
					tmp = m[i][j];

				if(tmp > max)	
					max = tmp;
			}
		}

		// 将 i+1 行至 n-1 行对应列的元素依次加到 i 行
		// 每加一行，就求一次当前叠加行的最大子数组和
		for(i=0; i<n-1; ++i)
		{
			for(j=i+1; j<n; ++j)
			{
				tmp = 0;
				for(k=0; k<n; ++k)
				{
					m[i][k] += m[j][k];
					if(tmp > 0)
						tmp += m[i][k];
					else
						tmp = m[i][k]; 

					if(tmp > max)	
						max = tmp;
				}
			}
		}

		cout << max << endl;
	}

}
```
